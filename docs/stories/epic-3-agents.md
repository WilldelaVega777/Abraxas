# Epic 3: Agent Orchestration & Chat
**Status:** Planned
**Priority:** High (P1)
**Phase:** 2

## Description
Enable the core value proposition: interacting with AI Agents directly from the IDE.

## User Stories Scope
1.  **Agent Chat Interface:** Build the chat UI in the Sidebar with support for markdown, code blocks, and tool call visualization.
2.  **Session Management:** Implement the logic to handle multiple isolated agent sessions (tabs).
3.  **Backend Connection:** Wire up the UI to the Python Backend via WebSockets for real-time streaming.
4.  **Agent Configuration:** Build the visual "Agent Designer" in the Management Console to create/edit agent personas.
