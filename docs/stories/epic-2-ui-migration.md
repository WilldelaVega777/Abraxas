# Epic 2: Neon Hex UI & UX Migration
**Status:** Ready for Dev
**Priority:** High (P1)
**Phase:** 1

## Description
Transform the existing Archon UI into the premium "Abraxas Neon Hex" design system and migrate it to the new Vite-based architecture.

## User Stories Scope
1.  **Vite Migration:** Port existing React components from Next.js to Vite, replacing `next/image` and `next/router`.
2.  **Design System:** Implement the "Neon Hex" theme tokens (colors, typography, shadows) in Tailwind.
3.  **Hexagonal Components:** Create custom SVG/Canvas components for Hexagonal Nodes and Avatars.
4.  **Sidebar UI:** Build the specific responsive layout for the VSCode Sidebar (Operations Center).
5.  **Management Console UI:** Adapt the main dashboard layout for the central editor WebView.
